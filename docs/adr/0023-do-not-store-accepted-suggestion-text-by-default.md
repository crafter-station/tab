# Do Not Store Accepted Suggestion Text By Default

Tab records acceptance metadata for product quality, but it does not store accepted suggestion text, Rewrite selected text, surrounding typing context, or final inserted text by default. Rewrite selected and replacement text are also ineligible for Memory Extraction by default because selection authorship is unknown. Any future raw-text debugging or evaluation capture must be explicit opt-in, visibly indicated, short-retained, and deletable.
