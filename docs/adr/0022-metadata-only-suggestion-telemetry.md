# Metadata Only Suggestion Telemetry

Tab may record suggestion interaction telemetry such as generated, shown, accepted, dismissed, stale, or failed outcome; trigger and inference mode; latency; application category; plan, model, memory-use, and cloud-cost metadata; and suggestion, selected-text, surrounding-text, and accepted word or character counts. It does not store raw typing context, Rewrite selected text, surrounding context, suggestion or replacement text, or final inserted text as telemetry by default. This supports product quality measurement without creating a backdoor typing log.
