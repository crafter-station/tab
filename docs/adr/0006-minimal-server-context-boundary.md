# Minimal Server Context Boundary

Tab's MVP sends only recent typing context or explicitly selected Rewrite text with bounded text before and after it, active application identity, eligible personalization, and client metadata to the suggestion API. Rewrite text is sent only after local sensitive-data screening; a secure field or any detected secret-like selected or surrounding text suppresses the request instead of transmitting redacted placeholders. Tab does not send raw key events, screenshots, full document or field contents, clipboard contents, browser URLs, window titles, or accessibility tree dumps, preserving a clear privacy boundary around what the native app may transmit.
